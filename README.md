# databricks-medallion-pipeline
Developed a strong foundation and hands-on experience in implementing Medallion Architecture (Bronze, Silver, Gold) in Databricks, covering incremental data ingestion, data transformation, and dimensional modeling concepts.
## Project Overview
This project demonstrates an end-to-end data pipeline using Databricks, following the Medallion Architecture approach:
- **Bronze Layer**: Ingests raw data with incremental loading logic
- **Silver Layer**: Cleans, transforms, and standardizes data
- **Gold Layer**: Creates business-ready datasets for analytics and reporting
## Technologies Used
- Databricks
- PySpark
- SQL
